# LiveScore API - Football Live Scores Backend

Une API REST complète pour le suivi des matchs de football en temps réel, avec scores, statistiques, joueurs, équipes, classements et gestion des favoris.

## 🎯 Caractéristiques

- ✅ Authentification JWT sécurisée
- ✅ Synchronisation en temps réel avec API-FOOTBALL
- ✅ Gestion complète des matchs (live, à venir, terminés)
- ✅ Profils équipes et joueurs détaillés
- ✅ Classements et statistiques
- ✅ Système de favoris utilisateur
- ✅ Documentation API automatique (Swagger)
- ✅ Base de données MySQL optimisée

---

## 📋 Stack Technique

| Composant | Technologie |
|-----------|------------|
| Backend | Python 3.11+ |
| Framework | FastAPI |
| Database | MySQL 8.0+ |
| ORM | SQLAlchemy 2.0 |
| Authentication | JWT (PyJWT) |
| Password Hash | bcrypt |
| HTTP Client | Requests |
| API External | api-football.com |
